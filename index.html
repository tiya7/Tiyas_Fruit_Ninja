Features

Built Different 🔥

A fully local, offline game that runs entirely on your machine — no internet required after setup.

🖐
Real Hand Tracking
MediaPipe AI tracks your index finger tip in real time at 60 FPS — no gloves, no markers, no wands.
🍓
15 Vibrant Fruits
Watermelon, banana, orange, kiwi, mango, grapes and more — all with hand-crafted photorealistic sprites.
💣
Bombs & Danger
One wrong swipe on a bomb and it's game over. Weave through the chaos with precision.
⚡
Combo System
Chain slices within 1 second to build wild combos. Your multiplier grows with every streak.
🏆
High Score Tracker
Your personal best is tracked live on-screen. Beat it and get a glowing "New Record!" celebration.
🚀
Fast Startup
Assets load in parallel, model is cached on first run. After setup, game starts in under 3 seconds.

Gameplay

How to Play

1
Open terminal and run
python -m tiya.main
2
Wait for the loading screen — hand tracker initialises automatically
3
Hold your index finger up in front of the camera
4
Move it over flying fruit to slice — the green dot shows your position
5
Chain slices fast for COMBOS — avoid the black bomb!
Controls

Keys & Actions

☝ Finger
Move over fruit to slice it
Q
Quit the game
Esc
Also quits
R
Restart after Game Over
Setup

Get Started in 5 Steps

Requires Python 3.9+ and a webcam. Works on Windows, Mac, and Linux.

1
Clone from GitHub
git clone https://github.com/tiya7/tiyas-fruit-ninja.git cd tiyas-fruit-ninja
2
Create a virtual environment
py -3 -m venv .venv .venv\Scripts\Activate.ps1
3
Install dependencies
pip install mediapipe==0.10.32 opencv-python numpy
4
Run the game
python -m tiya.main
5
🎉 Play! First run downloads the AI model (~8 MB) automatically.
